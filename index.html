Services

CommonLAK is a comprehensive Search Engine for the LAK Dataset. It provides services to easily explore, navigate and visualize the dataset.

EXPLORE

Search the Dataset

Search the Dataset effectively based on keywords. Get results in tabular format. A handy tool for the general public

NAVIGATE

Query the Dataset

Query the dataset directly using SPARQL and access the required information. Best suited for data researchers.

VISUALIZE

Picturize the Dataset

Simple visualizations using Bar plots, Graphs and other charts. Helps researchers and common people alike.
